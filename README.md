# Portfolio Franky Mbomeyo Essono

Portfolio professionnel moderne pour Franky Mbomeyo Essono, expert en droit des assurances au Gabon.

## Technologies

- **Next.js 16** - Framework React full-stack
- **TypeScript** - Typage statique
- **TailwindCSS** - Styles modernes
- **Framer Motion** - Animations fluides
- **shadcn/ui** - Composants UI

## Déploiement Gratuit

### Option 1 : Vercel (Recommandé)
1. Poussez sur GitHub
2. Connectez votre compte Vercel
3. Importez le dépôt GitHub
4. Déployez automatiquement

### Option 2 : Netlify
1. Build command: `npm run build`
2. Publish directory: `.next`
3. Node version: 18+

### Option 3 : Railway
1. Connectez GitHub
2. Choisissez le dépôt
3. Déployez avec Next.js template

## Développement Local

```bash
npm install
npm run dev
```

Ouvrez `http://localhost:3000`

## Variables d'Environnement

Optionnel - pour la gestion de documents :
```
BLOB_READ_WRITE_TOKEN=votre_token_vercel_blob
```

## Structure

- `/app` - Pages Next.js
- `/components` - Composants React
- `/public` - Assets statiques
- `/admin` - Interface d'administration
